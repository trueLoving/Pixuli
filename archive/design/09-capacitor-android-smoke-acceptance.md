## 1. 构建与安装

| 包类型       | 命令                         | APK 路径                                                        | 用途                             |
| ------------ | ---------------------------- | --------------------------------------------------------------- | -------------------------------- |
| Debug 离线   | `pnpm build:android:debug`   | `apps/pixuli/android/app/build/outputs/apk/debug/app-debug.apk` | **推荐冒烟**                     |
| Release 签名 | `pnpm build:android:release` | `.../release/app-release.apk`                                   | 发版前                           |
| Live Reload  | `pnpm dev:android`           | —                                                               | 开发联调，**不**作为离线签收依据 |

```bash
# 仓库根目录
pnpm build:android:debug
adb install -r apps/pixuli/android/app/build/outputs/apk/debug/app-debug.apk
```

Gitee 源使用直连 raw URL；本地工作区优先（见 PoC §4.3）。
