### ✨ 主要功能

Pixuli Mobile 是一款基于 React Native 构建的智能图片管理移动应用。

#### 🖼️ 图片管理

- 网格浏览：2列布局，懒加载，支持多种图片格式（JPEG, PNG, WebP, GIF, SVG, BMP）
- 上传删除：相机拍照/相册选择上传，批量删除
- 全屏预览：左右滑动浏览，底部缩略图导航
- 元数据：查看和编辑图片名称、尺寸、描述、标签等信息
- 下拉刷新：实时同步最新内容

#### 🎨 图片处理

- 压缩：可调节质量（10%-100%）
- 格式转换：支持 JPEG、PNG、WebP 格式转换
- 尺寸调整：自定义宽高，支持保持宽高比
- 图片裁剪：拖动裁剪框，调整大小和位置
- 处理预览：显示处理前后对比、文件大小、压缩率

#### ☁️ 云端存储

- 存储后端：GitHub 和 Gitee 集成
- 配置管理：支持配置导入/导出
- 版本控制：自动版本管理
- 性能优化：智能缓存元数据，提升加载速度

#### 🔍 搜索筛选

- 搜索：支持图片名称搜索
- 筛选：按标签、尺寸、日期筛选
- 排序：支持多种排序方式（名称、日期、大小等）

#### 📷 相机集成

- 拍照上传：直接拍照或从相册选择
- 拍照后编辑：即时编辑描述和标签
- 权限管理：自动请求相机和存储权限

#### 🎬 幻灯片播放

- 自动播放：可配置播放间隔和自动播放模式
- 播放模式：顺序播放和随机播放
- 过渡效果：淡入淡出、滑动、缩放等多种动画
- 循环播放：支持循环播放所有图片
- 播放控制：播放/暂停/停止、上一张/下一张
- 图片列表：侧边栏显示所有图片缩略图

#### 🎨 主题和国际化

- 主题切换：浅色/深色/自动主题
- 多语言：支持中文和英文界面，实时切换

### 📥 安装说明

#### Android 安装步骤

1. **下载 APK 文件**
   - 从 GitHub Release 页面下载 `Pixuli_{版本号}.apk` 文件（请替换为实际版本号）

2. **启用未知来源安装**
   - 打开 Android 设备的"设置"
   - 进入"安全"或"应用"设置
   - 启用"允许安装未知来源的应用"或"安装未知应用"选项

3. **安装应用**
   - 使用文件管理器打开下载的 APK 文件
   - 点击"安装"按钮
   - 按照提示完成安装

4. **首次配置**
   - 打开应用后，进入"设置"页面
   - 配置 GitHub 或 Gitee 存储后端
   - 输入必要的认证信息（Token、仓库名等）

### ⚠️ 注意事项

- **平台支持**：本版本仅支持 Android 平台，iOS 版本正在开发中
- **系统要求**：建议 Android 5.0 (API 21) 及以上版本
- **存储配置**：首次使用需要配置 GitHub 或 Gitee 存储后端
- **网络要求**：需要网络连接以访问云端存储服务
- **权限要求**：需要相机权限（用于拍照功能）和存储权限（用于访问相册）
