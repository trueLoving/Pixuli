### ✨ 主要功能

Pixuli Desktop 是一款基于 Electron 构建的跨平台智能图片管理桌面应用。

#### 🖼️ 图片管理

| 功能           | 描述                                       |
| -------------- | ------------------------------------------ |
| **智能浏览**   | 网格布局展示，支持懒加载，流畅浏览大量图片 |
| **拖拽上传**   | 支持单张和批量图片上传，直观的拖拽操作     |
| **批量操作**   | 批量上传和删除，实时显示操作进度           |
| **标签系统**   | 自定义标签，支持多标签管理和过滤           |
| **搜索功能**   | 按名称、描述、标签快速查找图片             |
| **元数据编辑** | 编辑图片名称、描述和标签                   |
| **全屏预览**   | 沉浸式全屏预览体验，支持缩放和旋转操作     |

#### 🔧 图片处理

| 功能          | 描述                                           |
| ------------- | ---------------------------------------------- |
| **WebP 压缩** | 可调节压缩质量（10%-100%），平衡质量和文件大小 |
| **格式转换**  | 支持 JPEG、PNG、WebP 三种格式相互转换          |
| **尺寸调整**  | 自定义宽度和高度，支持保持宽高比               |
| **批量处理**  | 批量压缩和格式转换，大幅提升工作效率           |
| **实时预览**  | 处理前预览效果，显示压缩前后对比               |
| **AI 分析**   | AI 图片分析，场景识别，自动标签生成            |

#### ☁️ 云端存储

| 功能            | 描述                                                    |
| --------------- | ------------------------------------------------------- |
| **GitHub 集成** | 使用 GitHub 仓库作为图片存储后端                        |
| **Gitee 集成**  | 支持 Gitee 仓库，为国内用户提供更快访问                 |
| **存储源切换**  | 灵活在 GitHub 和 Gitee 之间切换存储源                   |
| **配置管理**    | 支持配置的导入、导出和清除，方便迁移和备份              |
| **版本控制**    | 充分利用 Git 的版本控制功能，所有操作都有完整的历史记录 |

#### 🪟 多窗口模式

| 窗口类型        | 功能描述                   |
| --------------- | -------------------------- |
| **主窗口**      | 源管理和项目列表的主要界面 |
| **项目窗口**    | 独立的图片浏览和管理窗口   |
| **压缩窗口**    | 专门的图片压缩功能窗口     |
| **转换窗口**    | 独立的格式转换窗口         |
| **AI 分析窗口** | 独立的 AI 分析功能窗口     |

#### 📋 操作日志

| 功能         | 描述                               |
| ------------ | ---------------------------------- |
| **操作记录** | 自动记录所有上传、删除、编辑等操作 |
| **日志查看** | 便捷查看历史操作详情               |
| **日志导出** | 支持导出操作日志，方便备份和分析   |

#### ⌨️ 快捷键和用户体验

### 📥 安装说明

#### macOS 安装步骤

1. **下载安装包**
   - 从 GitHub Release 页面下载 `Pixuli-{版本号}-mac-{架构}.dmg` 文件
   - 支持 Intel (x64) 和 Apple Silicon (ARM64) 架构

2. **安装应用**
   - 双击下载的 DMG 文件
   - 将 Pixuli 应用拖拽到"应用程序"文件夹
   - 首次运行时，如果出现安全提示，请前往"系统设置" >
     "隐私与安全性" > 点击"仍要打开"

3. **首次配置**
   - 打开应用后，进入"设置"页面
   - 配置 GitHub 或 Gitee 存储后端
   - 输入必要的认证信息（Token、仓库名等）

#### Windows 安装步骤

1. **下载安装包**
   - 从 GitHub Release 页面下载 `Pixuli-Setup-{版本号}-x64.exe` 文件

2. **安装应用**
   - 双击下载的 EXE 文件
   - 按照安装向导提示完成安装
   - 首次运行时，如果出现 Windows Defender 安全提示，点击"更多信息" > "仍要运行"

3. **首次配置**
   - 打开应用后，进入"设置"页面
   - 配置 GitHub 或 Gitee 存储后端
   - 输入必要的认证信息（Token、仓库名等）

### ⚠️ 注意事项

- **平台支持**：本版本支持 macOS (x64, ARM64) 和 Windows
  (x64)，Linux 版本正在开发中
- **系统要求**：
  - macOS: macOS 10.13 及以上版本
  - Windows: Windows 10 及以上版本
- **存储配置**：首次使用需要配置 GitHub 或 Gitee 存储后端
- **网络要求**：需要网络连接以访问云端存储服务
- **权限要求**：需要文件系统访问权限（用于拖拽上传和本地文件操作）
