# ⌨️ Pixuli 键盘快捷键指南

> 掌握键盘快捷键，让图片管理更加高效！

Pixuli 提供了丰富的键盘快捷键支持，让您无需鼠标即可快速完成各种操作。无论是图片浏览、编辑还是管理，都能通过键盘实现流畅的操作体验。

## 🚀 快速开始

### 基础操作
- **F1** - 打开快捷键帮助界面
- **Esc** - 关闭当前对话框或模态框
- **F5** - 刷新图片列表
- **/** - 快速聚焦到搜索框

### 图片导航
- **↑↓←→** - 在图片间导航选择
- **Enter** - 打开选中的图片
- **Space** - 切换网格/列表视图模式

## 🎯 功能快捷键

### 核心功能
| 快捷键 | 功能 | 描述 |
|--------|------|------|
| **Ctrl + C** | 图片压缩 | 快速打开图片压缩工具 |
| **Ctrl + F** | 格式转换 | 打开图片格式转换功能 |
| **Ctrl + A** | AI 分析 | 启动 AI 图片分析 |
| **Ctrl + ,** | 设置 | 打开 GitHub 配置设置 |

### 视图切换
| 快捷键 | 功能 | 描述 |
|--------|------|------|
| **Ctrl + V** | 切换视图 | 在网格视图和列表视图间切换 |
| **Ctrl + +** | 放大 | 放大图片预览 |
| **Ctrl + -** | 缩小 | 缩小图片预览 |
| **Ctrl + 0** | 重置 | 重置图片预览大小 |

## 🖼️ 图片浏览技巧

### 键盘导航
使用方向键在图片间快速导航：

- **↑** - 选择上一张图片（网格视图：上一行）
- **↓** - 选择下一张图片（网格视图：下一行）  
- **←** - 选择左侧图片
- **→** - 选择右侧图片

### 视觉反馈
- 选中的图片会有 **蓝色边框** 和阴影效果
- 选中图片会 **轻微放大** 显示
- 右上角显示 **蓝色选中标记** ✓
- 自动滚动到选中的图片位置

## ❓ 常见问题

### Q: 快捷键不生效怎么办？
A: 请检查是否在输入框中，某些快捷键在输入状态下会被禁用。

### Q: 可以自定义快捷键吗？
A: 目前版本暂不支持自定义，但这是我们的开发计划之一。

### Q: 如何查看所有快捷键？
A: 按 **F1** 键即可打开完整的快捷键帮助界面。

### Q: 快捷键在不同系统上有差异吗？
A: 在 macOS 上，Ctrl 键对应 Cmd 键的功能。
