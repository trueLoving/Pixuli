## 1. 侧栏 IA（P0 · #196）

| Before                                      | After                                                                                  |
| ------------------------------------------- | -------------------------------------------------------------------------------------- |
| 工具/设置混在「功能」分组；工作区埋在设置内 | **图床 / 工具 / 工作区** 独立菜单；**设置** 为左右布局弹窗（同步、快捷键、版本、文档） |

## 2. 主内容区（P1）

| Before                                                   | After                                                                  |
| -------------------------------------------------------- | ---------------------------------------------------------------------- |
| `ImageContent` 内联错误条 + 桌面端重复「图片库 (n)」标题 | 统一 `ContentFeedback`（含重试）+ `ImageLibraryEmpty` + 工具栏卡片样式 |

| ------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| 网格 `ImageGridActions`、列表展开行、预览层各自一套按钮 | `@pixuli/ui` **`ImageActionMenu`**（icon-bar / dropdown / labeled-bar）+ 网格**右键菜单** |
